## 🧮 Python Calculator (Revised Version)

### 🔹 What I Changed

This project is an improved version of my first Python calculator.

---

### 🧱 Old Version
- Basic calculator with fixed operations (`a, b, c, d, e`)
- No loop (runs only once)
- Limited error handling
- Manual operation selection system
- Basic user interaction

---

### ⚡ New Version (Improved)
- Added **loop system** → user can calculate multiple times  
- Added **try-except** → handles invalid inputs  
- Added **division by zero handling**  
- Switched to **operator-based input** (`+, -, *, /, //`)  
- Used **match-case** for cleaner logic  
- Improved **user experience** (countdown, processing messages)  
- Added **exit option**
